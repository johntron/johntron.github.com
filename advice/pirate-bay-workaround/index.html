Pirate Bay Workaround

Sep 20th, 2009

Here’s a fix for the Pirate Bay torrents. The primary tracker over at The Pirate Bay doesn’t seem to be working. As a workaround, I’ve create a simple script to replace torrent files downloaded form TPB to work with one of their secondary trackers (tracker.openbittorrent.com).

To use the script, just upload the torrent you got from TPB, hit submit, and it’ll prompt you to save the converted file. The source code of the script can be found here for the paranoid types.
